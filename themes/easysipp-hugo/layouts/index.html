Overview

SIPp is an excellent open-source tool for SIP performance testing, but its command-line interface can be daunting for many users. easySIPp (also known as kSipP) bridges this gap by offering an intuitive, web-based GUI that makes SIP and VoIP testing accessible to everyone.

Built with Python (Django) for the backend and modern web technologies (JavaScript, HTML, CSS) for the frontend, easySIPp streamlines the entire testing workflow. Whether you're a VoIP tester, network engineer, or simply need to validate SIP functionality, easySIPp is designed to save you time and effort.

Get Started

Getting easySIPp up and running is quick and easy, thanks to Docker:


# Pull the Docker image
docker pull krndwr/ksipp

# Run the Docker container
docker run -dt --network host --name ksipp krndwr/ksipp

Once your container is up and running, open your web browser and navigate to:

http://localhost:8080/

You can also replace localhost with your Linux/Docker IP address if you are running it on a remote machine.
